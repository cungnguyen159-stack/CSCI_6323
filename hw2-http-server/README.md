## How to Run

1. Make sure Node.js is installed (v22.x)

2. Navigate to the project folder

3. Run: `node server.js`

4. Open browser to: http://localhost:3000

## Features Implemented

- ✅ Multiple GET routes (/, /about, /projects, /contact)

- ✅ JSON API endpoint (/api/data)

- ✅ POST request handling (/api/contact)

- ✅ Static file serving (HTML, CSS, images)

- ✅ Proper Content-Type headers

- ✅ 404 error handling

## Routes

- GET / - Home page

- GET /about - About me

- GET /projects - My projects

- GET /contact - Contact form

- GET /api/data - JSON data

- POST /api/contact - Form submission
